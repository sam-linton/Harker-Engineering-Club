Mechanical

Introduction

Mechanical Engineering involves the design, build, and assembly of structures. The term Computer Aided Design (CAD) is used to refer to software tools used to do this. In robotics, a few likely applications include:

  • Chassis - the basic structure of a robot that holds the body together
  • Motor holders
  • Wheels and axles
  • Cover
  • Battery case
  • Controller case

3D Printing

3D Printing is a great way to create components for robots or other devices. There are 3 steps to making a 3D printed part:

  1. Create a Model

    Create a 3D model using a CAD program such as TinkerCad or OnShape Output an .stl file.

  2. Create a Slice File

    Use a program such as Bambu Studio to convert the .stl file you created in step 1 into a .3mf file needed by the 3d printer.

  3. Print the Object

    Use a 3D printer such as the Bambu A1 mini to print the 3D model using the .3mf file.
